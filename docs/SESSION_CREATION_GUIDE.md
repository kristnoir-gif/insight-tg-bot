# 📱 Создание Telethon сессий

## Опция 1: Интерактивное создание (рекомендуется)

Запустите на вашем компьютере (где установлен Python):

```bash
cd /Users/kristina/kris_/bot_tg
python3 create_session_interactive.py
```

Скрипт спросит:
1. Номер телефона аккаунта (в формате: +79999999999)
2. Имя сессии (например: 211766470_telethon)

Затем вам придёт код верификации в Telegram - введите его в консоль.

**Повторите для каждого аккаунта:**
- `211766470_telethon` (для @211766470)
- `kristina_user` (для вашего основного аккаунта)

---

## Опция 2: Пакетное создание

1. Отредактируйте `batch_create_sessions.py`:

```python
ACCOUNTS = {
    "211766470_telethon": "+79999999999",  # Замените на реальный номер
    "kristina_user": "+79999999998",       # И на этот тоже
}
```

2. Запустите:

```bash
python3 batch_create_sessions.py
```

---

## После создания сессий

Файлы сессий будут созданы в текущей папке:
- `211766470_telethon.session`
- `kristina_user.session`

Скопируйте их в `/Users/kristina/kris_/bot_tg/` и запустите бот:

```bash
python3 main.py
```

---

## Для бота требуется минимум 1 рабочая сессия!

Если у вас только одна учётная запись:
1. Создайте сессию под этим аккаунтом
2. Скопируйте файл сессии 3 раза с разными именами:

```bash
cp user_session.session 211766470_telethon.session
cp user_session.session kristina_user.session
```

Это позволит боту использовать один аккаунт для всех трёх "клиентов" пула.

---

## Проблемы?

- **"AuthKeyDuplicatedError"**: Сессия используется одновременно в двух местах. Убедитесь, что Telegram Desktop закрыт.
- **"No code was provided"**: Вы не получили код? Проверьте спам в Telegram.
- **"database is locked"**: Сессия используется другим процессом. Закройте все экземпляры бота.
